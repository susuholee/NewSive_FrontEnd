# NewSive Frontend

> Next.js 기반 실시간 뉴스 & 커뮤니티 플랫폼 **NewSive**의 프론트엔드 프로젝트입니다.

NewSive는 실시간으로 발생하는 뉴스와 알림을 빠르게 확인하고,  
커뮤니티 기능을 통해 사용자 간 소통을 제공하는 **실시간 정보 기반 플랫폼**을 목표로 합니다.

본 레포지토리는 **NewSive 서비스의 프론트엔드(UI/UX 및 화면 흐름)** 를 담당하며,  
현재는 **백엔드 API 연동 전 단계로, 페이지 단위 기능 동작과 사용자 흐름 구현에 집중**하고 있습니다.

---

## 기술 스택
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Server State**: TanStack React Query
- **HTTP Client**: Axios

## 구현된 기능 (UI / UX 기준)
> 아래 기능들은 **백엔드 연동 전, 프론트엔드 단에서의 UI 및 동작 흐름 구현 상태**입니다.

### 뉴스
- 뉴스 리스트 페이지 UI
- 뉴스 상세 페이지 라우팅 구조
- 외부 뉴스 API 연동을 고려한 구조 분리

### 인증
- 로그인 / 회원가입 페이지 UI
- 전역 인증 상태 관리 구조
- Protected 페이지 접근 제어

### 커뮤니티 (채팅)
- 실시간 채팅 UI
- 메시지 입력 / 전송 / Enter 키 처리
- 로그인 유저 기준 내/남 메시지 구분
- 서버 없이 동작하는 채팅 UX 흐름 구현

### 사용자
- 마이페이지 UI (허브 역할)
- 계정 정보 수정 UI
- 닉네임 변경 시 전역 상태 반영 구조
- 친구 관리 페이지 UI

### 알림
- 알림 센터 UI
- 알림 설정 페이지 UI

### 공통
- 상단 네비게이션 UI
- 전역 유저 정보 표시
- 공통 레이아웃 및 컴포넌트 구조 정리

## Architecture Notes

- 페이지 단위 UI와 사용자 흐름을 우선 구현
- 전역 상태(Zustand)를 중심으로 인증 및 사용자 정보 관리
- API 연동을 고려하여 화면 로직과 데이터 접근 로직 분리
- 공통 컴포넌트는 `shared/components` 하위에서 관리

## 개발 방향

- 프론트엔드 기능 흐름을 기준으로 백엔드 API 설계 진행

- 실시간 기능(채팅, 알림)은 WebSocket 기반으로 확장
